D3.js in Observable

One way to overcome the requirement to have a Python web server running in order to see the output from a .csv file in D3.js is to use Obesrvable.

Observable's workflow is very similar to a Python Jupyter Notebook in that you can enter your D3.js code in cells and run each cell separately. Observable notebooks can also be "forked" and "starred" in the same way that you might use Github so it has a very familiar feeling to it.

One huge advantage of deploying your D3.js code in Observable is that, once you publish it on their website you can then create an iFrame link to your notebook which can be embedded in a standard HTML page! Even if you have used a .csv file (like I have in the example in the left hand pane) you do not need to run a Python web server to view it!
